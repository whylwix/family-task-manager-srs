# Техническое задание на разработку "Family Task Manager" (Семейный менеджер задач)

## 1. Наименование и область применения
**Наименование программы:** "Family Task Manager" (Семейный менеджер задач).

**Область применения:** Программа предназначена для организации и распределения семейных обязанностей, отслеживания выполнения домашних задач, повышения семейной организованности и мотивации через систему баллов. Приложение может использоваться на персональных компьютерах, смартфонах и планшетах.

## 2. Функциональные требования
### 2.1. Управление задачами
Программа должна предоставлять возможность:
- Создавать задачи с указанием названия, описания, ответственного. (#1)
- Отмечать задачи как выполненные с подтверждением. (#2)
- Редактировать и удалять задачи.
- Присваивать задачи категориям ("Домашние дела", "Дети", "Покупки").
- Устанавливать срок выполнения и приоритет (низкий, средний, высокий).

### 2.2. Управление семьей
Программа должна позволять:
- Добавлять членов семьи в систему с указанием имени и роли. (#3)
- Назначать задачи конкретным членам семьи.
- Просматривать задачи по ответственным.
- Видеть статистику выполнения для каждого члена семьи.

### 2.3. Система мотивации
Программа должна включать:
- Начисление баллов за выполненные задачи.
- Систему достижений и наград.
- Рейтинг членов семьи по баллам.
- Визуализацию прогресса через dashboard с графиками. (#10)

### 2.4. Отслеживание прогресса
Программа должна предоставлять:
- Историю выполненных задач за период.
- Статистику по каждому члену семьи.
- Возможность экспорта статистики в CSV/PDF.

## 3. Требования к надежности
- Программа должна автоматически сохранять все изменения данных.
- При запуске программа должна загружать все ранее сохраненные данные.
- Потеря данных при случайном закрытии приложения недопустима.
- Резервное копирование должно выполняться раз в сутки.

## 4. Требования к интерфейсу
- Интерфейс должен быть выполнен на русском языке.
- Интерфейс должен быть интуитивно понятным для пользователей от 8 лет.
- Должна поддерживаться тёмная и светлая темы.
- Навигация должна быть простой: главный экран → задачи → семья → статистика → награды.

## 5. Требования к техническому обеспечению
### 5.1. Клиентская часть:
- Операционная система: Windows 10+, macOS 10.15+, iOS 13+, Android 9+
- Браузеры: Chrome 85+, Firefox 80+, Safari 13+ (для веб-версии)
- Память (ОЗУ): Не менее 1 ГБ
- Экран: Минимальное разрешение 320×568 точек

### 5.2. Серверная часть (для синхронизации между устройствами):
- Поддержка REST API
- Шифрование передаваемых данных
- Авторизация по JWT токену
- База данных: PostgreSQL / SQLite

## 6. Требования к безопасности
- Каждая семья имеет изолированные данные.
- Родительский контроль: только администраторы семьи могут удалять членов.
- Данные не передаются рекламным сетям.
- Пароли хранятся в хэшированном виде.
- Защита от XSS и CSRF атак.
