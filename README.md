Техническое задание на разработку "Менеджера задач по дому" (Family Task Manager)
1. Наименование и область применения
Наименование программы: "Family Task Manager" (Менеджер задач по дому).
Область применения: Программа предназначена для семейного использования с целью распределения домашних обязанностей между членами семьи, отслеживания выполнения задач, повышения организованности и мотивации к выполнению домашних дел. Приложение может использоваться на персональных компьютерах, смартфонах и планшетах.

2. Функциональные требования
2.1. Управление задачами
Программа должна предоставлять пользователю возможность:

Создавать задачи с указанием названия, описания, ответственного члена семьи. (#1)
Отмечать задачи как выполненные. (#2)
Редактировать текст существующих задач.
Удалять задачи.
Присваивать задачи категории ("Кухня", "Уборка", "Покупки").
Устанавливать дедлайн и приоритет.
2.2. Распределение обязанностей
Программа должна позволять:

Добавлять членов семьи в систему с указанием имени и роли. (#3)
Назначать задачи конкретному члену семьи.
Просматривать задачи по ответственным.
Видеть загрузку каждого члена семьи.
2.3. Отслеживание прогресса
Программа должна предоставлять:

Визуализацию прогресса семьи через dashboard с графиками. (#10)
Историю выполненных задач за период.
Статистику по каждому члену семьи.
Возможность экспорта статистики.
3. Требования к надежности
Программа должна автоматически сохранять все изменения данных.
При запуске программа должна загружать все ранее сохраненные данные.
Потеря данных при случайном закрытии приложения недопустима.
Резервное копирование должно выполняться раз в сутки.
4. Требования к интерфейсу
Интерфейс должен быть выполнен на русском языке.
Интерфейс должен быть интуитивно понятным для пользователей от 10 лет.
Должна поддерживаться тёмная и светлая темы.
Навигация должна быть простой: главный экран → задачи → члены семьи → статистика.
5. Требования к техническому обеспечению
5.1. Клиентская часть:
Операционная система: Windows 10+, macOS 10.15+, iOS 13+, Android 9+
Браузеры: Chrome 85+, Firefox 80+, Safari 13+ (для веб-версии)
Память (ОЗУ): Не менее 512 МБ
Экран: Минимальное разрешение 320×568 точек
5.2. Серверная часть (для синхронизации между устройствами):
Поддержка REST API
Шифрование передаваемых данных
Авторизация по токену
6. Требования к безопасности
Каждая семья имеет изолированные данные.
Родительский контроль: только родители могут удалять членов семьи.
Данные не передаются рекламным сетям.
Пароли хранятся в хэшированном виде.
