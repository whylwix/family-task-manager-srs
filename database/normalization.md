# Процесс нормализации базы данных Family Task Manager

## 1. Исходные требования
Приложение должно хранить:
- Информацию о семьях и их членах
- Задачи с назначением ответственных
- Категории задач с баллами
- Историю начисления баллов
- Напоминания о задачах
